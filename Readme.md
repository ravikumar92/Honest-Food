This project contains two module
    1. front-end
    2.backend

To start front-end
1. go to honest-food-ui folder
2. run npm install
3. npm start


To start back-end
1. go to honest-food-backend folder
2. run npm install
3. npm run dev

Note the backend contains api(can be used through postman also)-
1. localhost:8080/locate/?location=<address>
